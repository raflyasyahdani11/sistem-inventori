# Sistem Inventori

Sistem inventori Diso

## Requirement

-   PHP version > 8.2.0
-   Node version > 16.15.1

## How To Run

Ketikan perintah berikut pada terminal

-   Duplikat file `.env.example`

```bash
cp .env.example .env
```

-   Install dependencies

```sh
composer install; npm install; npm run build
```

- Jalankan aplikasi
```sh
php artisan serve
```
